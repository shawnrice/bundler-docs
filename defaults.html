Bundler Defaults

For easy access, these defaults come with the bundler. If you want to use the bundler for a different asset, then look at the custom assets page.

AppleScript

  • _display: information for connected displays.
  • _file: file access / manipulation.
  • _keychain: keychain access.
  • _list: better list functionality.
  • _number: helpers for handling numbers.
  • _plist: plist parsing for Applescript.
  • _string: library of string functions.
  • _url: helpers to work with URLs.
  • qWorkflow: helper library for Alfred workflows.

Bash

  • BashWorkflowHandler: helper library for Alfred workflows.
  • shunit2: unit testing in Bash

PHP

  • CFPropertyList-php: plist parsing for PHP.
  • portable-utf8: solid utf8 support for PHP.
  • Workflows: David Ferguson's PHP library for Alfred.
  • Access to Packagist using Composer

Python

Access to all PyPi packages via Pip.

Ruby

Acess to Ruby Gems.

Utilities

  • cocoaDialog: create native OS X dialogs.
  • GlyphManager: alter workflow icons in Alfred.
  • notificator: send notifications.
  • Pashua: create advanced native OS X dialogs.
  • SetupIconsForTheme: alter workflow icons in Alfred.
  • Terminal-Notifier: send notifications.
  • viewer: view local or remote webpages in a modal window.
