Custom Assets


{
    "name": "ASSETNAME",
    "type": "TYPE",
    "description": "short description — all lowercase, no end punctuation",
    "versions": {
        "latest": {
            "invoke": "COMMAND",
            "files": [
                {
                    "url": "URL",
                    "method": "METHOD"
                }
            ],
            "install": [
                "cp -f '__CACHE__/FILE' '__DATA__'"
            ]
        }
    }
}
methods: direct zip tgz
Note: the name of the asset must match the filename, which also must the be name that you call the asset with. So, if you have custom-asset as the name, then the filename must be custom-asset.json, and you must load the asset with $b->load('type', 'custom-asset', 'version', '/path/to/json');
